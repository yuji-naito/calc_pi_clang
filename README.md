以降に記載するビルド、実行方法は Ubuntu で確認したもの。

## ビルド方法

```bash
$ gcc -o calc_pi_clang main.c
```

## 実行方法

```bash
$ ./calc_pi_clang
```

## 出力結果の例

count: 点の数
pi: 円周率の計算結果
time: 処理時間[s]

```bash
count:  100000000       pi: 3.1415226800        time: 1.5111840000
count: 1000000000       pi: 3.1415427080        time: 15.1256300000
```
